# quiz-app

一般常識クイズアプリ。

## 技術スタック

- HTML / CSS / JavaScript (バニラ)
- フレームワーク・ビルドツールなし

## プロジェクト構成

```
quiz-app2/
├── index.html        # エントリーポイント
├── style.css         # スタイル
├── script.js         # アプリロジック
└── questions.js      # クイズ問題データ
```

## 開発方針

- バニラJS で実装する。外部ライブラリは原則使用しない。
- HTML/CSS/JS の各ファイルは役割ごとに分離する。
- スマートフォン対応（レスポンシブ）を考慮する。

## 機能概要

- 一般常識に関するクイズを出題する。
- 選択肢形式（4択）で回答する。
- 正誤判定・スコア集計を行う。
- 全問終了後に結果を表示する。
